# 개발전 사진 촬영

          train 폴더에 마스크를 정상적으로 착용한 사진, 마스크를 착용 하지 않은 사진, 마스크를 코까지만 착용한 사진을 찍어서
          폴더에 저장했으며 사진의 갯수는 학습 후 결과의 오차를 줄이기 위해 많은 사진을 넣는것이 중요하다.
